# UI de l'app en anglais : les chaînes visibles suivent le README public

Toutes les chaînes **visibles par l'utilisateur** de l'app island — cartes de Session (labels d'état, outil, contexte, tâches de fond, facts), Peek, jauges de Quotas, barre des menus, notifications macOS — sont rédigées **en anglais**. C'est la suite directe d'ADR-0011 (README public EN) : island s'adresse aux utilisateurs de Claude Code, anglophones par défaut, et une UI moitié française au milieu d'une vitrine internationale trahissait ce public. Décidé au fil de l'issue #115. Comme pour ADR-0011, tout le reste est inchangé : `CONTEXT.md`, PRD, ADR, issues et scénarios `docs/test-scenarios/` restent **en français** ; code, identifiants et messages de commit restent en anglais.

## Consequences

- **Lexique d'état unique** partagé par les cartes, le Peek et le menu, aligné sur la Priorité d'état (`waiting > terminé > working > idle`, CONTEXT.md) : `.idle → started`, `.running → working`, `.ended → done`, `.waiting → waiting`. Un seul mot par état, jamais deux synonymes selon la surface.
- **Vocabulaire produit traduit de façon stable**, comme le README : `Réponse depuis l'Island → « Answer from the Island »`, `Quotas`/`statusline` conservés. Les listes _Avoid_ de `CONTEXT.md` s'appliquent aussi en anglais (ADR-0011) : le Liseré devient « Edge outline » — jamais *glow*, *halo* ni *border* (interdits) ; le README a fixé « outline » pour cette surface.
- **Typographie anglaise** : les pourcentages perdent l'espace insécable à la française (`63 % → 63%`, contexte comme jauges) ; la fenêtre hebdomadaire passe de `7 j` à `7 d` (`5 h` inchangé) ; les guillemets `« … »` du dernier prompt deviennent des guillemets droits `"…"` ; l'espace avant deux-points disparaît (`outil : X → tool: X`).
- **Frontière domaine vs UI.** Les noms d'**état de domaine** de `CONTEXT.md` (démarrée, en cours, terminée, en attente) restent français dans le code interne, les commentaires, les **traces stdout** (canal développeur/agentique) et les descriptions de tests ; seuls les **libellés rendus** sont traduits. Les tests agentiques asserttent les tokens de trace (`=waiting`, `=ended`, `×1bg`), qui sont les noms de cas `SessionState` (déjà anglais), donc insensibles à cette traduction. Seul HP-04, qui épinglait un libellé visible en vérification visuelle, a vu sa chaîne attendue passée à l'anglais (sa prose reste FR).
- **Décision peu réversible** dans le même sens qu'ADR-0011 : revenir en arrière = re-traduire l'UI et re-piéger les tests de présentation. L'exception à la règle docs-FR (CLAUDE.md règle 2) est donc actée ici, comme le README l'a été dans ADR-0011 ; la règle 2 n'est pas réécrite.
